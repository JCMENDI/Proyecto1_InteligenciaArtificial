# Proyecto 1 - Clasificación de Solicitudes a Mesa de Ayuda

Clasificador de tickets de soporte al cliente implementado con Naive Bayes Multinomial desde cero, sin librerías de ML automáticas.


**Curso:** Inteligencia Artificial - Primer Semestre 2026
**Universidad:** Rafael Landívar - Facultad de Ingeniería

## Integrantes:
- José Carlos Mendizábal Huertas 1077222
- Luis Fernando Samayoa Sagastume

## Descripción
El sistema recibe una solicitud de soporte escrita en lenguaje natural y la clasifica automáticamente en una de 11 categorías (ORDER, BILLING, SHIPPING, REFUND, ACCOUNT, etc.), simulando el enrutamiento automático de tickets a los departamentos correspondientes.

## Tecnologías
- Python 3.10+
- Flask (servidor web)
- NLTK (solo para tokenización y stopwords)
- HTML + CSS + JavaScript + Bootstrap (frontend)
- Pickle (persistencia del modelo)
